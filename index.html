Apa itu geo.BlankOn?

geo.BlankOn adalah merupakan kegiatan gotong royong digital untuk mengembangkan gugus perangkat lunak dan data geospasial yang dibuat untuk mendukung beragam platform komputer.

Sistem geo.BlankOn dibuat generik dan dapat dikustomisasi untuk tema tertentu (wisata, kesehatan, pendapatan daerah, bencana, dan sebagainya). Basis geo.BlankOn dipublikasikan dalam lisensi bebas.

Klien

Klien geo.BlankOn adalah aplikasi-aplikasi yang menampilkan data geospasial secara visual di komputer atau perangkat bergerak. Saat ini aplikasi klien geo.BlankOn yang bernama Peta terpasang secara bawaan pada BlankOn Suroboyo.

Aplikasi ini dapat dibangun dan dijalankan untuk perangkat bergerak yang menggunakan sistem Android, iOS, maupun BlackBerry 10.

Server

Aplikasi server menyediakan layanan geo.BlankOn di Internet berupa:

  • fitur (Point-Of-Interest, ruang bangun, area, dan sebagainya)
  • peta dasar
  • rute

Aplikasi ini nantinya akan dapat diakses secara pemrograman melalui jalur API. Saat ini aplikasi server masih dalam tahap konsep pengembangan.

Data

Data geo.BlankOn merupakan semua data yang dapat dipublikasikan melalui layanan geo.BlankOn di server dengan lisensi bebas.

Saat ini, kegiatan di bidang data belum dimulai, ini merupakan kesempatan Anda untuk bergabung dan memimpin area ini

geo.BlankOn membutuhkan segala bantuan dari Anda yang memiliki minat dalam bidang geospasial dalam konteks perangkat lunak bebas/terbuka.
